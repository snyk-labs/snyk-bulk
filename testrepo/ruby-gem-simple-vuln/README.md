# Ruby::Simple::Vuln::Gem

A simple vulnerable gem

## Installation

Add this line to your application's Gemfile:

```ruby
gem 'ruby-simple-vuln-gem'
```

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install ruby-simple-vuln-gem
